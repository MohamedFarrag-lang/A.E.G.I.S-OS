<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">«"Color is not decoration. Color is visual

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">- Tint</span>

<span dir="ltr">- Shade</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="rtl">تمر الألوان داخل المشروع بالمراحل التالية:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- الضوء الطبيعي أو

<span dir="ltr"></span><span dir="rtl">- معالجة الصورة داخل

<span dir="ltr">4. LUTs</span>

<span dir="ltr"></span>

<span dir="rtl">يشير</span> <span dir="ltr">LUT (Look-Up
Table</span><span dir="rtl">) إلى ملف يحتوي على تعليمات رياضية لتحويل

<span dir="rtl">تُستخدم</span> <span dir="ltr">LUTs</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Types of LUTs</span>

<span dir="ltr"></span>

<span dir="ltr">Technical LUTs</span>

<span dir="ltr"></span>

<span dir="rtl">تُستخدم لتحويل ملفات التصوير (مثل</span>
<span dir="ltr">Log</span> <span dir="rtl">أو</span>
<span dir="ltr">RAW</span><span dir="rtl">) إلى مساحة ألوان قياسية

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">LUT Best Practices</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- ابدأ دائمًا بالتصحيح اللوني قبل

<span dir="ltr"></span><span dir="rtl">- لا تعتمد على</span>
<span dir="ltr">LUT</span> <span dir="rtl">لإصلاح أخطاء التصوير.</span>

<span dir="ltr"></span><span dir="rtl">- اضبط شدة تأثير</span>

<span dir="ltr"></span><span dir="rtl">- استخدم</span>
<span dir="ltr">LUTs</span> <span dir="rtl">موثوقة ومتوافقة مع مساحة
الألوان.</span>

<span dir="ltr"></span><span dir="rtl">- اختبر النتيجة على أكثر من
شاشة.</span>

<span dir="ltr"></span>
