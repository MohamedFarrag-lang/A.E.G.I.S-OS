<span dir="ltr"></span>

<span dir="ltr">4. Graph Editor</span>

<span dir="ltr"></span>

<span dir="rtl">يُعد</span> <span dir="ltr">Graph Editor</span>
<span dir="rtl">أحد أهم الأدوات في برامج تصميم الحركة، لأنه يمنح المصمم
تحكمًا دقيقًا في سرعة الحركة وتسارعها وتباطؤها، بدلاً من الاعتماد على
الحركة الخطية الافتراضية.</span>

<span dir="ltr"></span>

<span dir="rtl">لا يغيّر</span> <span dir="ltr">Graph Editor</span>
<span dir="rtl">موقع العنصر فقط، بل يغيّر سلوك الحركة، وهو ما يجعل
الأنيميشن يبدو طبيعيًا واحترافيًا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Why Graph Editor Matters</span>

<span dir="ltr"></span>

<span dir="rtl">يساعد</span> <span dir="ltr">Graph Editor</span>
<span dir="rtl">على:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- التحكم في سرعة الحركة.</span>

<span dir="ltr"></span><span dir="rtl">- إنشاء حركات أكثر واقعية.</span>

<span dir="ltr"></span><span dir="rtl">- إزالة الإحساس بالتحريك
الآلي.</span>

<span dir="ltr"></span><span dir="rtl">- تحسين الإيقاع البصري.</span>

<span dir="ltr"></span><span dir="rtl">- بناء انتقالات سلسة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Types of Graphs</span>

<span dir="ltr"></span>

<span dir="ltr">Value Graph</span>

<span dir="ltr"></span>

<span dir="rtl">يعرض تغير قيمة الخاصية مع الزمن.</span>

<span dir="ltr"></span>

<span dir="rtl">يستخدم مع خصائص مثل:</span>

<span dir="ltr"></span>

<span dir="ltr">- Position</span>

<span dir="ltr">- Scale</span>

<span dir="ltr">- Rotation</span>

<span dir="ltr">- Opacity</span>

<span dir="ltr"></span>

<span dir="rtl">ويساعد على فهم كيفية تغير القيمة أثناء الحركة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Speed Graph</span>

<span dir="ltr"></span>

<span dir="rtl">يعرض سرعة العنصر أثناء الحركة.</span>

<span dir="ltr"></span>

<span dir="rtl">يعتبر الأكثر استخدامًا في</span> <span dir="ltr">Motion
Graphics</span> <span dir="rtl">لأنه يسمح بالتحكم في:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- التسارع.</span>

<span dir="ltr"></span><span dir="rtl">- التباطؤ.</span>

<span dir="ltr"></span><span dir="rtl">- سرعة البداية.</span>

<span dir="ltr"></span><span dir="rtl">- سرعة النهاية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Graph Editing Techniques</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل أهم أساليب العمل داخل</span> <span dir="ltr">Graph
Editor</span><span dir="rtl">:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- تعديل</span>
<span dir="ltr">Bezier Handles</span><span dir="rtl">.</span>

<span dir="ltr"></span><span dir="rtl">- إنشاء منحنيات سلسة.</span>

<span dir="ltr"></span><span dir="rtl">- التحكم في سرعة البداية.</span>

<span dir="ltr"></span><span dir="rtl">- التحكم في سرعة النهاية.</span>

<span dir="ltr"></span><span dir="rtl">- إزالة الحركات المفاجئة.</span>

<span dir="ltr"></span><span dir="rtl">- موازنة الإيقاع بين
العناصر.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Best Practices</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- تجنب الحركة الخطية في معظم
الحالات.</span>

<span dir="ltr"></span><span dir="rtl">- اجعل المنحنيات ناعمة
وطبيعية.</span>

<span dir="ltr"></span><span dir="rtl">- حافظ على اتساق الحركة داخل
المشروع.</span>

<span dir="ltr"></span><span dir="rtl">- راقب سرعة الحركة وليس المسافة
فقط.</span>

<span dir="ltr"></span><span dir="rtl">- استخدم المعاينة المستمرة أثناء
التعديل.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">5. Easing</span>

<span dir="ltr"></span>

<span dir="rtl">يشير</span> <span dir="ltr">Easing</span>
<span dir="rtl">إلى الطريقة التي تبدأ بها الحركة أو تنتهي بها.</span>

<span dir="ltr"></span>

<span dir="rtl">في الواقع، لا تتحرك الأجسام بسرعة ثابتة، ولذلك فإن
استخدام</span> <span dir="ltr">Easing</span> <span dir="rtl">يجعل الحركة
أكثر إقناعًا وواقعية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Types of Easing</span>

<span dir="ltr"></span>

<span dir="ltr">Ease In</span>

<span dir="ltr"></span>

<span dir="rtl">تبدأ الحركة ببطء ثم تتسارع تدريجيًا.</span>

<span dir="ltr"></span>

<span dir="rtl">يستخدم عند:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- انطلاق العناصر.</span>

<span dir="ltr"></span><span dir="rtl">- بداية الانتقالات.</span>

<span dir="ltr"></span><span dir="rtl">- تحريك الكاميرا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Ease Out</span>

<span dir="ltr"></span>

<span dir="rtl">تبدأ الحركة بسرعة ثم تتباطأ قبل التوقف.</span>

<span dir="ltr"></span>

<span dir="rtl">يستخدم عند:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- توقف العناصر.</span>

<span dir="ltr"></span><span dir="rtl">- انتهاء الحركة.</span>

<span dir="ltr"></span><span dir="rtl">- وصول العنصر إلى هدفه.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Ease In Out</span>

<span dir="ltr"></span>

<span dir="rtl">تبدأ الحركة ببطء، ثم تتسارع، ثم تتباطأ مرة أخرى.</span>

<span dir="ltr"></span>

<span dir="rtl">وهو الخيار الأكثر استخدامًا في التصميم الاحترافي لأنه
يمنح الحركة انسيابية طبيعية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Benefits of Easing</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- يجعل الحركة أكثر طبيعية.</span>

<span dir="ltr"></span><span dir="rtl">- يزيد من راحة المشاهدة.</span>

<span dir="ltr"></span><span dir="rtl">- يحسن تجربة المستخدم.</span>

<span dir="ltr"></span><span dir="rtl">- يمنع الإحساس بالحركة
الآلية.</span>

<span dir="ltr"></span><span dir="rtl">- يعزز جودة الأنيميشن.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Best Practices</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- لا تستخدم الحركة الخطية إلا عند
الحاجة.</span>

<span dir="ltr"></span><span dir="rtl">- اجعل نوع الـ</span>
<span dir="ltr">Easing</span> <span dir="rtl">متوافقًا مع طبيعة
العنصر.</span>

<span dir="ltr"></span><span dir="rtl">- حافظ على اتساق منحنيات
الحركة.</span>

<span dir="ltr"></span><span dir="rtl">- اختبر الحركة بسرعات
مختلفة.</span>

<span dir="ltr"></span><span dir="rtl">- استخدم</span>
<span dir="ltr">Ease In Out</span> <span dir="rtl">كخيار افتراضي في أغلب
الحالات.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">6. Typography Animation</span>

<span dir="ltr"></span>

<span dir="rtl">يقصد بـ</span> <span dir="ltr">Typography
Animation</span> <span dir="rtl">تحريك النصوص بطريقة تدعم الرسالة وتجذب
انتباه المشاهد دون التأثير على سهولة القراءة.</span>

<span dir="ltr"></span>

<span dir="rtl">النص المتحرك ليس عنصرًا زخرفيًا فقط، بل وسيلة لنقل
المعلومات وإبراز الأفكار المهمة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Types of Text Animation</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل أكثر الأساليب استخدامًا:</span>

<span dir="ltr"></span>

<span dir="ltr">- Fade Animation</span>

<span dir="ltr">- Slide Animation</span>

<span dir="ltr">- Scale Animation</span>

<span dir="ltr">- Rotation Animation</span>

<span dir="ltr">- Mask Reveal</span>

<span dir="ltr">- Character Animation</span>

<span dir="ltr">- Word Animation</span>

<span dir="ltr">- Line Animation</span>

<span dir="ltr">- Kinetic Typography</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Typography Principles</span>

<span dir="ltr"></span>

<span dir="rtl">عند تحريك النصوص يجب مراعاة:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- وضوح القراءة.</span>

<span dir="ltr"></span><span dir="rtl">- التوقيت المناسب.</span>

<span dir="ltr"></span><span dir="rtl">- البساطة.</span>

<span dir="ltr"></span><span dir="rtl">- التناسق مع هوية المشروع.</span>

<span dir="ltr"></span><span dir="rtl">- تجنب الحركة المبالغ
فيها.</span>

<span dir="ltr"></span><span dir="rtl">- الحفاظ على تركيز
المشاهد.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Best Practices</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- استخدم خطوطًا واضحة.</span>

<span dir="ltr"></span><span dir="rtl">- اجعل مدة ظهور النص
مناسبة.</span>

<span dir="ltr"></span><span dir="rtl">- لا تحرك جميع العناصر في الوقت
نفسه.</span>

<span dir="ltr"></span><span dir="rtl">- حافظ على اتساق أسلوب
الحركة.</span>

<span dir="ltr"></span><span dir="rtl">- اجعل الحركة تخدم المحتوى لا أن
تطغى عليه.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">7. Shape Animation</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد</span> <span dir="ltr">Shape Animation</span>
<span dir="rtl">على تحريك الأشكال الهندسية لإنشاء واجهات ورسوم متحركة
مرنة وعالية الجودة.</span>

<span dir="ltr"></span>

<span dir="rtl">تتميز طبقات الأشكال بأنها قابلة للتعديل دون فقدان
الجودة، مما يجعلها الخيار المفضل في تصميم الشعارات والرسومات
التوضيحية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Common Shape Types</span>

<span dir="ltr"></span>

<span dir="ltr">- Rectangle</span>

<span dir="ltr">- Circle</span>

<span dir="ltr">- Ellipse</span>

<span dir="ltr">- Polygon</span>

<span dir="ltr">- Star</span>

<span dir="ltr">- Path</span>

<span dir="ltr">- Custom Shapes</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Shape Animation Techniques</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل التقنيات الأساسية:</span>

<span dir="ltr"></span>

<span dir="ltr">- Position Animation</span>

<span dir="ltr">- Scale Animation</span>

<span dir="ltr">- Rotation Animation</span>

<span dir="ltr">- Trim Paths</span>

<span dir="ltr">- Repeater</span>

<span dir="ltr">- Offset Paths</span>

<span dir="ltr">- Merge Paths</span>

<span dir="ltr">- Shape Morphing</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Best Practices</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- استخدم الأشكال المتجهية
(</span><span dir="ltr">Vector Shapes</span><span dir="rtl">) كلما
أمكن.</span>

<span dir="ltr"></span><span dir="rtl">- اجعل الحركة بسيطة
وواضحة.</span>

<span dir="ltr"></span><span dir="rtl">- حافظ على اتساق الألوان
والأحجام.</span>

<span dir="ltr"></span><span dir="rtl">- تجنب التعقيد غير
الضروري.</span>

<span dir="ltr"></span><span dir="rtl">- نظم الطبقات داخل
المشروع.</span>

<span dir="ltr"></span>

<span dir="ltr">8. Motion Systems</span>

<span dir="ltr"></span>

<span dir="rtl">يشير</span> <span dir="ltr">Motion System</span>
<span dir="rtl">إلى مجموعة القواعد والمعايير التي تنظم جميع الحركات داخل
المشروع لضمان الاتساق والجودة.</span>

<span dir="ltr"></span>

<span dir="rtl">يُستخدم هذا المفهوم في:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- التطبيقات.</span>

<span dir="ltr"></span><span dir="rtl">- المواقع الإلكترونية.</span>

<span dir="ltr"></span><span dir="rtl">- الهوية البصرية.</span>

<span dir="ltr"></span><span dir="rtl">- الإعلانات.</span>

<span dir="ltr"></span><span dir="rtl">- أنظمة التصميم
(</span><span dir="ltr">Design Systems</span><span dir="rtl">).</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Components of a Motion System</span>

<span dir="ltr"></span>

<span dir="rtl">يتكون النظام الحركي عادةً من:</span>

<span dir="ltr"></span>

<span dir="ltr">- Motion Guidelines</span>

<span dir="ltr">- Timing Standards</span>

<span dir="ltr">- Duration Rules</span>

<span dir="ltr">- Easing Standards</span>

<span dir="ltr">- Transition Library</span>

<span dir="ltr">- Animation Patterns</span>

<span dir="ltr">- Interaction Feedback</span>

<span dir="ltr">- Motion Tokens</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Benefits of Motion Systems</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- توحيد أسلوب الحركة.</span>

<span dir="ltr"></span><span dir="rtl">- تسريع عملية الإنتاج.</span>

<span dir="ltr"></span><span dir="rtl">- تحسين تجربة المستخدم.</span>

<span dir="ltr"></span><span dir="rtl">- تقليل الأخطاء.</span>

<span dir="ltr"></span><span dir="rtl">- تسهيل التعاون بين أعضاء
الفريق.</span>

<span dir="ltr"></span><span dir="rtl">- الحفاظ على هوية المشروع.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Motion System Principles</span>

<span dir="ltr"></span>

<span dir="ltr">- Consistency First.</span>

<span dir="ltr">- Purpose Before Animation.</span>

<span dir="ltr">- Simplicity Wins.</span>

<span dir="ltr">- Accessibility Matters.</span>

<span dir="ltr">- Performance Is Essential.</span>

<span dir="ltr">- Reusability Saves Time.</span>

<span dir="ltr"></span>

<span dir="rtl">يضمن الالتزام بهذه المبادئ أن تكون جميع الحركات داخل
المشروع متناسقة، واضحة، وسهلة التطوير والصيانة.</span>
