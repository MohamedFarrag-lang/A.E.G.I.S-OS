<span dir="ltr"></span>

<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل</span> <span dir="ltr">Motion Graphics</span>
<span dir="rtl">نقطة الالتقاء بين التصميم والحركة، حيث تتحول العناصر
البصرية الثابتة إلى محتوى ديناميكي قادر على جذب الانتباه، وتوضيح
المعلومات، وتعزيز الهوية البصرية، وإيصال الرسائل بطريقة أكثر
تأثيرًا.</span>

<span dir="ltr"></span>

<span dir="rtl">يهدف هذا الفصل إلى تأسيس فهم احترافي لمبادئ تصميم
الحركة، وكيفية إنشاء الرسوم المتحركة باستخدام منهجية منظمة تعتمد على
التوقيت، والحركة الطبيعية، والإيقاع البصري، بما يتوافق مع معايير صناعة
الأفلام، والإعلانات، والهوية البصرية، وصناعة المحتوى الرقمي.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">«"Motion is not decoration. Motion communicates
meaning."»</span>

<span dir="ltr"></span>

<span dir="rtl">لا تُستخدم الحركة لإضافة الجمال فقط، بل لتوجيه انتباه
المشاهد، وشرح المعلومات، وبناء التسلسل البصري، وتعزيز تجربة المستخدم.
الحركة الاحترافية هي التي تبدو طبيعية، وتدعم الرسالة، دون أن تشتت انتباه
الجمهور.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">1. Motion Design Fundamentals</span>

<span dir="ltr"></span>

<span dir="rtl">يشير</span> <span dir="ltr">Motion Design</span>
<span dir="rtl">إلى فن تصميم العناصر البصرية المتحركة بهدف إيصال
المعلومات أو بناء هوية بصرية أو دعم السرد القصصي.</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد تصميم الحركة على دمج عدة تخصصات، منها:</span>

<span dir="ltr"></span>

<span dir="ltr">- Graphic Design</span>

<span dir="ltr">- Animation</span>

<span dir="ltr">- Typography</span>

<span dir="ltr">- Visual Storytelling</span>

<span dir="ltr">- Color Theory</span>

<span dir="ltr">- Composition</span>

<span dir="ltr">- Timing</span>

<span dir="ltr">- Audio Design</span>

<span dir="ltr"></span>

<span dir="rtl">يختلف</span> <span dir="ltr">Motion Graphics</span>
<span dir="rtl">عن الرسوم المتحركة التقليدية، إذ يركز على تحريك النصوص،
والأشكال، والشعارات، والرسومات التوضيحية، والواجهات البصرية، بدلاً من
تحريك الشخصيات.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="rtl">أهداف</span> <span dir="ltr">Motion Graphics</span>

<span dir="ltr"></span>

<span dir="rtl">يساعد تصميم الحركة على:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- جذب انتباه المشاهد.</span>

<span dir="ltr"></span><span dir="rtl">- توضيح المعلومات المعقدة.</span>

<span dir="ltr"></span><span dir="rtl">- تعزيز الهوية البصرية.</span>

<span dir="ltr"></span><span dir="rtl">- تحسين تجربة المستخدم.</span>

<span dir="ltr"></span><span dir="rtl">- دعم السرد البصري.</span>

<span dir="ltr"></span><span dir="rtl">- زيادة التفاعل مع
المحتوى.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="rtl">أنواع</span> <span dir="ltr">Motion Graphics</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل التطبيقات الاحترافية لتصميم الحركة:</span>

<span dir="ltr"></span>

<span dir="ltr">- Logo Animation</span>

<span dir="ltr">- Title Sequences</span>

<span dir="ltr">- Lower Thirds</span>

<span dir="ltr">- Explainer Videos</span>

<span dir="ltr">- Infographics Animation</span>

<span dir="ltr">- Social Media Motion Graphics</span>

<span dir="ltr">- UI Motion Design</span>

<span dir="ltr">- Broadcast Graphics</span>

<span dir="ltr">- Advertising Motion Graphics</span>

<span dir="ltr"></span>

<span dir="rtl">لكل نوع متطلبات وأساليب تصميم مختلفة، لكن جميعها تعتمد
على المبادئ الأساسية للحركة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">2. Animation Principles</span>

<span dir="ltr"></span>

<span dir="rtl">تعتمد جميع أعمال</span> <span dir="ltr">Motion
Graphics</span> <span dir="rtl">الاحترافية على مبادئ التحريك التي تجعل
الحركة تبدو طبيعية ومقنعة.</span>

<span dir="ltr"></span>

<span dir="rtl">من أهم هذه المبادئ:</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Timing</span>

<span dir="ltr"></span>

<span dir="rtl">يحدد سرعة الحركة ومدتها، ويؤثر بشكل مباشر في الإحساس
بالوزن والطاقة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Spacing</span>

<span dir="ltr"></span>

<span dir="rtl">المسافة بين مواقع العنصر أثناء الحركة، وهي التي تحدد ما
إذا كانت الحركة تبدو سريعة أو بطيئة أو سلسة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Anticipation</span>

<span dir="ltr"></span>

<span dir="rtl">حركة تمهيدية تسبق الحركة الأساسية، وتساعد المشاهد على
توقع ما سيحدث.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Follow Through</span>

<span dir="ltr"></span>

<span dir="rtl">استمرار بعض أجزاء العنصر في الحركة بعد توقف الحركة
الرئيسية، مما يضيف واقعية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Overlapping Action</span>

<span dir="ltr"></span>

<span dir="rtl">تحرك أجزاء مختلفة من العنصر بسرعات مختلفة بدلاً من تحركها
جميعًا في الوقت نفسه.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Slow In & Slow Out</span>

<span dir="ltr"></span>

<span dir="rtl">تبدأ الحركة تدريجيًا، ثم تتسارع، ثم تتباطأ قبل التوقف،
وهو ما يجعلها أكثر طبيعية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Secondary Action</span>

<span dir="ltr"></span>

<span dir="rtl">إضافة حركة فرعية تدعم الحركة الأساسية دون أن تشتت
الانتباه.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Appeal</span>

<span dir="ltr"></span>

<span dir="rtl">يجب أن تكون الحركة جذابة وواضحة ومريحة بصريًا مهما كان
أسلوب التصميم.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="rtl">تطبيق هذه المبادئ هو ما يميز الحركة الاحترافية عن الحركة
الآلية أو الجامدة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">3. Keyframes</span>

<span dir="ltr"></span>

<span dir="rtl">تُعد</span> <span dir="ltr">Keyframes</span>
<span dir="rtl">الأساس الذي تعتمد عليه جميع برامج التحريك مثل</span>
<span dir="ltr">Adobe After Effects</span>
<span dir="rtl">و</span><span dir="ltr">Blender</span>
<span dir="rtl">وغيرها.</span>

<span dir="ltr"></span>

<span dir="rtl">الـ</span> <span dir="ltr">Keyframe</span>
<span dir="rtl">هو نقطة زمنية يتم فيها تحديد قيمة معينة لخاصية من خصائص
العنصر، ليقوم البرنامج بحساب الحركة بين هذه النقاط تلقائيًا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Common Animated Properties</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل الخصائص الأكثر استخدامًا:</span>

<span dir="ltr"></span>

<span dir="ltr">- Position</span>

<span dir="ltr">- Scale</span>

<span dir="ltr">- Rotation</span>

<span dir="ltr">- Opacity</span>

<span dir="ltr">- Anchor Point</span>

<span dir="ltr">- Skew</span>

<span dir="ltr">- Blur</span>

<span dir="ltr">- Effects Parameters</span>

<span dir="ltr"></span>

<span dir="rtl">يمكن تحريك خاصية واحدة أو عدة خصائص في الوقت نفسه لإنشاء
حركة أكثر تعقيدًا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">How Keyframes Work</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد التحريك على وجود نقطتين أساسيتين على الأقل:</span>

<span dir="ltr"></span>

<span dir="ltr">- Start Keyframe</span><span dir="rtl">: يحدد الحالة
الابتدائية.</span>

<span dir="ltr">- End Keyframe</span><span dir="rtl">: يحدد الحالة
النهائية.</span>

<span dir="ltr"></span>

<span dir="rtl">يقوم البرنامج بإنشاء الحركة بين هاتين النقطتين تلقائيًا،
ويمكن تعديلها لاحقًا باستخدام أدوات التحكم في المنحنيات والإيقاع.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Keyframe Types</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل الأنواع الأكثر استخدامًا:</span>

<span dir="ltr"></span>

<span dir="ltr">- Linear Keyframe</span>

<span dir="ltr">- Bezier Keyframe</span>

<span dir="ltr">- Continuous Bezier</span>

<span dir="ltr">- Auto Bezier</span>

<span dir="ltr">- Hold Keyframe</span>

<span dir="ltr">- Easy Ease</span>

<span dir="ltr"></span>

<span dir="rtl">اختيار نوع الـ</span> <span dir="ltr">Keyframe</span>
<span dir="rtl">المناسب يؤثر بشكل مباشر على طبيعة الحركة
وسلاستها.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Best Practices</span>

<span dir="ltr"></span>

<span dir="rtl">للحصول على حركة احترافية:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- ابدأ بتحريك العناصر الأساسية
أولًا.</span>

<span dir="ltr"></span><span dir="rtl">- تجنب إضافة عدد كبير من
الـ</span> <span dir="ltr">Keyframes</span> <span dir="rtl">دون
حاجة.</span>

<span dir="ltr"></span><span dir="rtl">- حافظ على اتساق التوقيت بين
العناصر.</span>

<span dir="ltr"></span><span dir="rtl">- استخدم أسماء واضحة
للطبقات.</span>

<span dir="ltr"></span><span dir="rtl">- راجع الحركة باستمرار أثناء
العمل.</span>

<span dir="ltr"></span><span dir="rtl">- لا تعتمد على الحركة الافتراضية
إذا كانت لا تخدم التصميم.</span>
