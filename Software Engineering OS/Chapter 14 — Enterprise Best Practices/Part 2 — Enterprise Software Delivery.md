<span dir="ltr"></span>

<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يمثل</span> <span dir="ltr">Enterprise Software
Delivery</span> <span dir="rtl">الإطار المؤسسي لتطوير وتسليم البرمجيات
على نطاق واسع داخل المؤسسات الكبرى. ويهدف إلى ضمان أن تنتقل الفكرة من
مرحلة التخطيط إلى منتج يعمل في بيئة الإنتاج بطريقة منظمة، وآمنة، وعالية
الجودة، مع المحافظة على سرعة التطوير واستقرار الأنظمة.</span>

<span dir="ltr"></span>

<span dir="rtl">في</span> <span dir="ltr">A.E.G.I.S.
OS</span><span dir="rtl">، لا يُنظر إلى تسليم البرمجيات كمرحلة نهائية، بل
كعملية مستمرة تبدأ منذ تحديد المتطلبات وتنتهي بالمراقبة والتحسين بعد
الإطلاق.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">«"البرمجيات المؤسسية الناجحة لا
تُقاس بسرعة تطويرها فقط، بل بسرعة تسليمها مع الحفاظ على الجودة، والأمان،
والاستقرار."»</span>
