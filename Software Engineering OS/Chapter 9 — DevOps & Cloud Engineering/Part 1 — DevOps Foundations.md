<span dir="ltr">Final Principles</span>

<span dir="ltr"></span>

<span dir="rtl">كل مشروع</span> <span dir="ltr">DevOps
Foundations</span> <span dir="rtl">داخل</span>
<span dir="ltr">A.E.G.I.S. OS</span> <span dir="rtl">يجب أن يلتزم

<span dir="ltr"></span>

<span dir="ltr">- Automate Everything Possible</span> <span dir="rtl">—
أتمتة كل ما يمكن أتمتته.</span>

<span dir="ltr">- Infrastructure is Software</span> <span dir="rtl">—
البنية التحتية تُعامل كشيفرة.</span>

<span dir="ltr">- Deploy with Confidence</span> <span dir="rtl">— يجب أن

<span dir="ltr">- Measure Before Improving</span> <span dir="rtl">— لا
يتم تحسين أي نظام دون قياس.</span>

<span dir="ltr">- Security is Continuous</span> <span dir="rtl">— الأمن
عملية مستمرة وليس مرحلة منفصلة.</span>

<span dir="ltr">- Version Everything</span> <span dir="rtl">— جميع

<span dir="ltr">- Feedback Drives Improvement</span> <span dir="rtl">—

<span dir="ltr">- Collaborate Without Silos</span> <span dir="rtl">—

<span dir="ltr">- Reliability Before Speed</span> <span dir="rtl">—
الاعتمادية أهم من سرعة النشر.</span>

<span dir="ltr">- Engineer for Continuous Delivery</span>
<span dir="rtl">— بناء أنظمة جاهزة للتسليم المستمر.</span>
