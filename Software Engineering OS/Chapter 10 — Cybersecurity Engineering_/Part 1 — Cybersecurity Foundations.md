<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يمثل</span> <span dir="ltr">Cybersecurity
Engineering</span> <span dir="rtl">التخصص الهندسي المسؤول عن تصميم وبناء
وتشغيل الأنظمة الرقمية بطريقة تضمن حماية البيانات، والأنظمة، والبنية

<span dir="ltr"></span>

حياة النظام، بدءًا من التصميم وحتى التشغيل والإدارة.</span>

<span dir="ltr"></span>

<span dir="ltr">1. Cybersecurity Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">Definition</span>

<span dir="ltr"></span>

<span dir="rtl">الأمن السيبراني هو مجموعة من المبادئ، والعمليات،
والتقنيات التي تهدف إلى حماية الأنظمة الرقمية من الوصول غير المصرح به،
أو التعديل، أو التعطيل، أو السرقة، مع ضمان استمرارية الأعمال.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Objectives</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- حماية البيانات.</span>

<span dir="ltr"></span><span dir="rtl">- حماية المستخدمين.</span>

<span dir="ltr"></span><span dir="rtl">- ضمان استمرارية الخدمات.</span>

<span dir="ltr"></span><span dir="rtl">- تقليل المخاطر.</span>

<span dir="ltr"></span><span dir="rtl">- دعم الامتثال والحوكمة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Security Mindset</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد مهندس الأمن على افتراض أن التهديدات ستحدث، ولذلك
يُبنى النظام ليكون قادرًا على المقاومة، والاكتشاف، والاستجابة،
والتعافي.</span>

<span dir="ltr"></span>

<span dir="ltr">5. Security Principles</span>

<span dir="ltr"></span>

<span dir="ltr">Least Privilege</span>

<span dir="ltr"></span>

<span dir="rtl">منح أقل مستوى من الصلاحيات اللازمة لتنفيذ المهمة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Defense in Depth</span>

<span dir="ltr"></span>

<span dir="rtl">استخدام عدة طبقات من الحماية بدلاً من الاعتماد على وسيلة
واحدة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Zero Trust</span>

<span dir="ltr"></span>

<span dir="rtl">عدم الثقة بأي مستخدم أو جهاز بشكل افتراضي، مع التحقق
المستمر من الهوية والصلاحيات.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Fail Secure</span>

<span dir="ltr"></span>

<span dir="rtl">عند حدوث خطأ، يجب أن ينتقل النظام إلى حالة آمنة وليس إلى
حالة مكشوفة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Secure by Design</span>

<span dir="ltr"></span>

<span dir="rtl">إدماج الأمن في التصميم منذ البداية، وليس إضافته
لاحقًا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Simplicity</span>

<span dir="ltr"></span>

<span dir="rtl">كلما كان التصميم أبسط، كان أكثر سهولة في المراجعة
والحماية.</span>
