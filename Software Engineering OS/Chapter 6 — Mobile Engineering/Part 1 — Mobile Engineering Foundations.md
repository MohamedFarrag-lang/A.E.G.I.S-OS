<span dir="ltr"></span>

<span dir="ltr">Final Principles</span>

<span dir="ltr"></span>

<span dir="rtl">كل مشروع</span> <span dir="ltr">Mobile
Engineering</span> <span dir="rtl">داخل</span>
<span dir="ltr">A.E.G.I.S. OS</span> <span dir="rtl">يجب أن يلتزم
بالمبادئ التالية:</span>

<span dir="ltr"></span>

<span dir="ltr">- User Experience First</span> <span dir="rtl">— تجربة
المستخدم هي المعيار الأساسي.</span>

<span dir="ltr">- Performance by Design</span> <span dir="rtl">— الأداء
جزء من التصميم وليس تحسينًا لاحقًا.</span>

<span dir="ltr">- Respect the Platform</span> <span dir="rtl">— احترام
معايير وإرشادات</span> <span dir="ltr">Android</span>
<span dir="rtl">و</span><span dir="ltr">iOS</span><span dir="rtl">.</span>

<span dir="ltr">- Secure Mobile Applications</span> <span dir="rtl">—
حماية البيانات والاتصالات في جميع الطبقات.</span>

<span dir="ltr">- Efficient Resource Usage</span> <span dir="rtl">—
الاستخدام الأمثل للبطارية، والذاكرة، والمعالج، والشبكة.</span>

<span dir="ltr">- Architecture Before Features</span> <span dir="rtl">—
تبدأ كل ميزة بمعمارية واضحة.</span>

<span dir="ltr">- Maintainable Codebase</span> <span dir="rtl">— قاعدة
كود منظمة وقابلة للتطوير.</span>

<span dir="ltr">- Test Early, Test Continuously</span> <span dir="rtl">—
الاختبار المستمر جزء من التطوير.</span>

<span dir="ltr">- Document Critical Decisions</span> <span dir="rtl">—
توثيق القرارات الهندسية المهمة.</span>

<span dir="ltr">- Engineer for Longevity</span> <span dir="rtl">— بناء
تطبيقات تستمر في التطور دون الحاجة إلى إعادة كتابة كاملة.</span>

<span dir="ltr">Closing Statement</span>

<span dir="ltr"></span>

<span dir="ltr">Mobile Engineering</span> <span dir="rtl">هو علم تصميم

والأمان، وسهولة الاستخدام، وقابلية التوسع. ويختلف عن تطوير الويب في
اعتماده على موارد محدودة، ودورات حياة خاصة بالتطبيق، ومتطلبات صارمة لكل
منصة.</span>
