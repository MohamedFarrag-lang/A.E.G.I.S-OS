<span dir="ltr"></span>

<span dir="ltr">\# Overview</span> <span dir="rtl">(نظرة عامة)</span>

<span dir="ltr"></span>

<span dir="rtl">يُعد \*\*</span><span dir="ltr">Website & Application
Design System</span><span dir="rtl">\*\* المرجع الرسمي لتصميم جميع مواقع
وتطبيقات \*\*</span><span dir="ltr">CORE X</span><span dir="rtl">\*\*،
ويهدف إلى إنشاء تجربة استخدام موحدة، وواجهة احترافية، ونظام تصميم قابل
لإعادة الاستخدام والتطوير.</span>

<span dir="ltr"></span>

<span dir="rtl">هذا النظام يضمن أن جميع المنتجات الرقمية التابعة
لـ</span> <span dir="ltr">CORE X</span> <span dir="rtl">تستخدم نفس
الهوية البصرية، ونفس المكونات، ونفس قواعد التصميم، مما يوفر تجربة مستخدم
متناسقة وسهلة.</span>

<span dir="ltr"></span>

<span dir="rtl">إنشاء نظام تصميم احترافي، مرن، وقابل للتوسع، يساعد على
بناء منتجات رقمية عالية الجودة بسرعة وكفاءة مع الحفاظ على هوية</span>
<span dir="ltr">CORE X</span><span dir="rtl">.</span>
