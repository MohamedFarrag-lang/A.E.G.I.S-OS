<span dir="ltr">\## 1. Responsive Design</span>

<span dir="ltr"></span>

<span dir="ltr">Responsive Design</span> <span dir="rtl">هو أسلوب تصميم
يجعل الواجهة تتكيف تلقائيًا مع مختلف أحجام الشاشات.</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد على:</span>

<span dir="ltr"></span>

<span dir="ltr">• Fluid Layouts</span>

<span dir="ltr">• Flexible Elements</span>

<span dir="ltr">• Responsive Images</span>

<span dir="ltr">• Relative Units</span>

<span dir="ltr">• Media Queries</span>

<span dir="ltr">• Scalable Components</span>

<span dir="ltr"></span>

<span dir="rtl">الهدف هو توفير تجربة استخدام متسقة عبر جميع
الأجهزة.</span>

<span dir="ltr"></span>
