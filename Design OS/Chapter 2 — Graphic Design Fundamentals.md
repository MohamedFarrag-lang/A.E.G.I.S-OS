<span dir="ltr"></span>

<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل \*\*</span><span dir="ltr">Graphic Design
Fundamentals</span><span dir="rtl">\*\* المرجع الأساسي للتصميم الجرافيكي

الفهم.</span>

<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">\*\*"Graphic design is visual communication with
purpose."\*\*</span>

<span dir="ltr"></span>

<span dir="rtl">كل عنصر داخل التصميم يجب أن يخدم الرسالة، ولا يوجد عنصر

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 1. Graphic Design</span>

<span dir="ltr"></span>

<span dir="rtl">التصميم الجرافيكي هو فن وعلم التواصل البصري باستخدام
العناصر الرسومية.</span>

<span dir="ltr"></span>

<span dir="rtl">يشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Branding</span>

<span dir="ltr">• Advertising</span>

<span dir="ltr">• Print Design</span>

<span dir="ltr">• Social Media Design</span>

<span dir="ltr">• Packaging</span>

<span dir="ltr">• Editorial Design</span>

<span dir="ltr">• Digital Graphics</span>

<span dir="ltr">• Marketing Materials</span>

<span dir="ltr"></span>

<span dir="rtl">المصمم الجيد يحول الأفكار إلى رسائل بصرية مؤثرة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 2. Visual Hierarchy</span>

<span dir="ltr"></span>

<span dir="rtl">الترتيب البصري يساعد المستخدم على فهم المحتوى
بسرعة.</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد على:</span>

<span dir="ltr"></span>

<span dir="ltr">• Size</span>

<span dir="ltr">• Color</span>

<span dir="ltr">• Contrast</span>

<span dir="ltr">• Typography</span>

<span dir="ltr">• Position</span>

<span dir="ltr">• Spacing</span>

<span dir="ltr"></span>

<span dir="rtl">العناصر الأكثر أهمية يجب أن تكون الأكثر وضوحًا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 3. Layout</span>

<span dir="ltr"></span>

<span dir="rtl">الـ</span> <span dir="ltr">Layout</span>
<span dir="rtl">هو تنظيم العناصر داخل مساحة التصميم.</span>

<span dir="ltr"></span>

<span dir="rtl">يشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Grid Systems</span>

<span dir="ltr">• Margins</span>

<span dir="ltr">• Columns</span>

<span dir="ltr">• Sections</span>

<span dir="ltr">• Structure</span>

<span dir="ltr">• Flow</span>

<span dir="ltr"></span>

<span dir="rtl">التخطيط الجيد يجعل التصميم أكثر وضوحًا وتنظيمًا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 4. Composition</span>

<span dir="ltr"></span>

<span dir="rtl">التكوين هو الطريقة التي يتم بها توزيع العناصر داخل
التصميم.</span>

<span dir="ltr"></span>

<span dir="rtl">يشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Focal Point</span>

<span dir="ltr">• Rule of Thirds</span>

<span dir="ltr">• Visual Flow</span>

<span dir="ltr">• Symmetry</span>

<span dir="ltr">• Asymmetry</span>

<span dir="ltr">• Depth</span>

<span dir="ltr"></span>

<span dir="rtl">التكوين الجيد يقود عين المشاهد بشكل طبيعي.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 5. Balance</span>

<span dir="ltr"></span>

<span dir="rtl">التوازن يمنح التصميم الاستقرار البصري.</span>

<span dir="ltr"></span>

<span dir="rtl">أنواعه:</span>

<span dir="ltr"></span>

<span dir="ltr">• Symmetrical Balance</span>

<span dir="ltr">• Asymmetrical Balance</span>

<span dir="ltr">• Radial Balance</span>

<span dir="ltr"></span>

<span dir="rtl">التوازن لا يعني التماثل، بل تحقيق الانسجام بين
العناصر.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 6. Contrast</span>

<span dir="ltr"></span>

<span dir="rtl">التباين يساعد على إبراز العناصر المهمة.</span>

<span dir="ltr"></span>

<span dir="rtl">يمكن تحقيقه باستخدام:</span>

<span dir="ltr"></span>

<span dir="ltr">• Color</span>

<span dir="ltr">• Size</span>

<span dir="ltr">• Shape</span>

<span dir="ltr">• Weight</span>

<span dir="ltr">• Space</span>

<span dir="ltr">• Typography</span>

<span dir="ltr"></span>

<span dir="rtl">التباين يزيد من وضوح التصميم وسهولة قراءته.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 7. Alignment</span>

<span dir="ltr"></span>

<span dir="rtl">المحاذاة تنظم العلاقة بين العناصر.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Left Alignment</span>

<span dir="ltr">• Center Alignment</span>

<span dir="ltr">• Right Alignment</span>

<span dir="ltr">• Grid Alignment</span>

<span dir="ltr">• Baseline Alignment</span>

<span dir="ltr"></span>

<span dir="rtl">المحاذاة الصحيحة تجعل التصميم أكثر احترافية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 8. Repetition</span>

<span dir="ltr"></span>

<span dir="rtl">التكرار يعزز الاتساق والهوية البصرية.</span>

<span dir="ltr"></span>

<span dir="rtl">يشمل تكرار:</span>

<span dir="ltr"></span>

<span dir="ltr">• Colors</span>

<span dir="ltr">• Typography</span>

<span dir="ltr">• Shapes</span>

<span dir="ltr">• Icons</span>

<span dir="ltr">• Patterns</span>

<span dir="ltr">• Components</span>

<span dir="ltr"></span>

<span dir="rtl">الاتساق البصري يبني هوية قوية ويسهل فهم التصميم.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="rtl">المساحات البيضاء هي الفراغات بين العناصر.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">• Reduce Clutter</span>

<span dir="ltr">• Increase Focus</span>

<span dir="ltr">• Create Balance</span>

<span dir="ltr">• Enhance Elegance</span>

<span dir="ltr">• Organize Content</span>

<span dir="ltr"></span>

<span dir="rtl">المساحات الفارغة عنصر تصميمي مهم وليست مساحة
مهدرة.</span>
