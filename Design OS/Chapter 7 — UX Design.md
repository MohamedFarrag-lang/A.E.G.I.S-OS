<span dir="ltr"></span>

<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل \*\*</span><span dir="ltr">UX
Design</span><span dir="rtl">\*\* المرجع الأساسي لتصميم تجربة المستخدم
داخل \*\*</span><span dir="ltr">Design OS</span><span dir="rtl">\*\*،
حيث يركز على فهم احتياجات المستخدمين، وتحليل سلوكهم، وتصميم رحلات
استخدام فعالة، وتنظيم المعلومات، وتحسين سهولة الاستخدام، وبناء منتجات
رقمية تحقق أهداف المستخدم والأعمال في آنٍ واحد.</span>

<span dir="ltr"></span>

<span dir="rtl">تجربة المستخدم لا تتعلق بالشكل، بل بكيفية شعور المستخدم
أثناء استخدام المنتج، ومدى سهولة إنجاز مهامه بكفاءة ورضا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">\*\*"User Experience is not what users see. It is what
users feel."\*\*</span>

<span dir="ltr"></span>

<span dir="rtl">أفضل تجربة مستخدم هي التي تجعل تحقيق الأهداف يبدو طبيعيًا
وسهلًا دون تعقيد.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">\## 1. UX Fundamentals</span>

<span dir="ltr"></span>

<span dir="ltr">UX Design</span> <span dir="rtl">هو عملية تصميم وتحسين
التجربة الكاملة للمستخدم أثناء التفاعل مع المنتج.</span>

<span dir="ltr"></span>

<span dir="rtl">يرتكز على:</span>

<span dir="ltr"></span>

<span dir="ltr">• User Needs</span>

<span dir="ltr">• Business Goals</span>

<span dir="ltr">• Usability</span>

<span dir="ltr">• Accessibility</span>

<span dir="ltr">• Functionality</span>

<span dir="ltr">• Satisfaction</span>

<span dir="ltr"></span>

<span dir="rtl">الهدف هو تحقيق تجربة بسيطة، فعالة، وممتعة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 2. User Research</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">• Surveys</span>

<span dir="ltr">• Observation</span>

<span dir="ltr">• Analytics</span>

<span dir="ltr">• Competitor Analysis</span>

<span dir="ltr">• User Feedback</span>

<span dir="ltr"></span>

<span dir="rtl">لا يمكن تصميم تجربة ناجحة دون فهم المستخدم
الحقيقي.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

\*\*</span><span dir="ltr">Design OS</span><span dir="rtl">\*\* العقلية

<span dir="ltr"></span>

قراراتك.</span>

<span dir="ltr"></span><span dir="rtl">• طور التجربة باستمرار.</span>
