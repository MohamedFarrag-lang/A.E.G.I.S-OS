<span dir="ltr">Chapter 6 — UI Design</span>

<span dir="ltr"></span>

<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل \*\*</span><span dir="ltr">UI
Design</span><span dir="rtl">\*\* المرجع الأساسي لتصميم واجهات المستخدم
داخل \*\*</span><span dir="ltr">Design OS</span><span dir="rtl">\*\*،
حيث يركز على بناء واجهات احترافية تجمع بين الوضوح، والجمال، وسهولة
الاستخدام، والاتساق، وقابلية التوسع، من خلال استخدام أنظمة التصميم
الحديثة، والمكونات القابلة لإعادة الاستخدام،
و</span><span dir="ltr">Design Tokens</span><span dir="rtl">، بما يضمن
تجربة مستخدم عالية الجودة.</span>

<span dir="ltr"></span>

<span dir="rtl">واجهة المستخدم ليست مجرد شاشة جميلة، بل هي نقطة التواصل
المباشرة بين الإنسان والمنتج، وكل عنصر فيها يجب أن يخدم هدفًا
واضحًا.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">\*\*"Great UI is invisible. Users notice the experience,
not the interface."\*\*</span>

<span dir="ltr"></span>

<span dir="rtl">واجهة المستخدم الاحترافية تجعل المستخدم يركز على إنجاز
المهمة، وليس على كيفية استخدام الواجهة.</span>

<span dir="ltr"></span>

<span dir="ltr">\## 1. UI Fundamentals</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">• Forms</span>

<span dir="ltr">• Cards</span>

<span dir="ltr">• Inputs</span>

<span dir="ltr">• Menus</span>

<span dir="ltr">• Modals</span>

الاستخدام.</span>
