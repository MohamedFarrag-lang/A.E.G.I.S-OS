<span dir="ltr">\# Overview</span> <span dir="rtl">(نظرة عامة)</span>

<span dir="ltr"></span>

<span dir="rtl">يُعد \*\*</span><span dir="ltr">CORE X Brand
Guidelines</span><span dir="rtl">\*\* المرجع الرسمي لاستخدام الهوية
البصرية لعلامة \*\*</span><span dir="ltr">CORE
X</span><span dir="rtl">\*\*، ويهدف إلى توحيد جميع عناصر العلامة

<span dir="ltr"></span>

<span dir="rtl">سواء كنت تصمم موقعًا إلكترونيًا، أو تطبيقًا، أو منشورًا على

المرجع الوحيد الذي يجب الرجوع إليه.</span>

<span dir="ltr">\# Mission</span> <span dir="rtl">(الرسالة)</span>

<span dir="ltr"></span>

<span dir="rtl">إنشاء هوية بصرية احترافية، موحدة، قابلة للتوسع، وسهلة
التطبيق، تعكس رؤية</span> <span dir="ltr">CORE X</span>
<span dir="rtl">وتضمن ظهور العلامة التجارية بنفس الجودة في جميع
الاستخدامات.</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- حماية العلامة التجارية.</span>

<span dir="ltr"></span><span dir="rtl">- تسهيل العمل بين المصممين

الهوية.</span>

مستقبلًا.</span>
