<span dir="ltr">\# Overview</span> <span dir="rtl">(نظرة عامة)</span>

<span dir="ltr"></span>

<span dir="rtl">يُعد \*\*</span><span dir="ltr">Cross
References</span><span dir="rtl">\*\* النظام المسؤول عن ربط جميع أجزاء
\*\*</span><span dir="ltr">A.E.G.I.S. OS</span><span dir="rtl">\*\*
ببعضها البعض بطريقة منطقية ومنظمة، بحيث لا يوجد أي</span>
<span dir="ltr">Chapter</span> <span dir="rtl">أو</span>
<span dir="ltr">Document</span> <span dir="rtl">أو</span>
<span dir="ltr">Template</span> <span dir="rtl">أو</span>
<span dir="ltr">SOP</span> <span dir="rtl">يعمل بمعزل عن باقي
النظام.</span>

<span dir="ltr"></span>

الصلة، مما يحول</span> <span dir="ltr">A.E.G.I.S. OS</span>
<span dir="rtl">إلى \*\*</span><span dir="ltr">Knowledge
Network</span><span dir="rtl">\*\* بدلاً من مجموعة ملفات منفصلة.</span>

<span dir="ltr">\# Mission</span> <span dir="rtl">(الرسالة)</span>

<span dir="ltr"></span>

(</span><span dir="ltr">Connected Knowledge
