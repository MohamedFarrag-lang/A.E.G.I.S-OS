<span dir="ltr"></span>

<span dir="ltr">\# Overview</span>

<span dir="ltr"></span>

<span dir="ltr">\*\*A.E.G.I.S. CORE</span><span dir="rtl">\*\* هو القلب
المركزي لنظام \*\*</span><span dir="ltr">A.E.G.I.S.
OS</span><span dir="rtl">\*\*، ويُعد المرجع الأعلى
(</span><span dir="ltr">Highest Authority</span><span dir="rtl">) الذي
يحكم جميع أنظمة التشغيل (</span><span dir="ltr">Operating
Systems</span><span dir="rtl">)، والمجلدات، والفصول، والوثائق، والقوالب،
والإجراءات (</span><span dir="ltr">SOPs</span><span dir="rtl">)،
والـ</span> <span dir="ltr">Master Prompts</span> <span dir="rtl">داخل
المشروع.</span>

<span dir="ltr"></span>

<span dir="rtl">لا يجوز إنشاء أو تعديل أو حذف أي جزء من</span>
<span dir="ltr">A.E.G.I.S. OS</span> <span dir="rtl">إلا إذا كان متوافقًا
مع القواعد والمعايير والسياسات الموجودة داخل</span>
<span dir="ltr">A.E.G.I.S. CORE</span><span dir="rtl">.</span>

<span dir="ltr"></span>

<span dir="rtl">يضمن هذا المجلد توحيد جميع الأنظمة، ومنع التكرار،
والحفاظ على الجودة، وقابلية التوسع، واستمرارية التطوير.</span>

<span dir="ltr"></span>

<span dir="ltr">\# Mission</span>

<span dir="ltr"></span>

<span dir="rtl">إنشاء مرجع مركزي موحد يضمن أن جميع أجزاء</span>
<span dir="ltr">A.E.G.I.S. OS</span> <span dir="rtl">تعمل وفق نفس
الفلسفة، ونفس المعايير، ونفس أساليب التنفيذ، ليصبح النظام قابلاً للتوسع،
وسهل الصيانة، وقابلاً للتطوير على المدى الطويل.</span>
