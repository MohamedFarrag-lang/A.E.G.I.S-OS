<span dir="ltr"></span>

<span dir="ltr">\# Overview</span> <span dir="rtl">(نظرة عامة)</span>

<span dir="ltr"></span>

<span dir="rtl">يُعد \*\*</span><span dir="ltr">Versioning
System</span><span dir="rtl">\*\* أحد أهم الأنظمة داخل
\*\*</span><span dir="ltr">A.E.G.I.S. CORE</span><span dir="rtl">\*\*،
حيث يتحكم في كيفية إنشاء الإصدارات، وإدارة التعديلات، وتوثيق التطوير،
والحفاظ على تاريخ المشروع بالكامل.</span>

<span dir="ltr"></span>

<span dir="rtl">الهدف من هذا النظام هو ضمان أن كل تغيير يتم داخل
\*\*</span><span dir="ltr">A.E.G.I.S. OS</span><span dir="rtl">\*\* يكون
موثقًا، ويمكن تتبعه، ويمكن الرجوع إليه في أي وقت، دون فقدان أي نسخة
سابقة.</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد النظام على مبدأ:</span>

<span dir="ltr"></span>

<span dir="ltr">\> \*\*"Nothing is ever lost. Everything is
versioned."\*\*</span>

<span dir="ltr"></span>

<span dir="rtl">أي:</span>

<span dir="ltr"></span><span dir="rtl">\> \*\*لا يتم حذف أي إصدار نهائي
أبدًا، وإنما يتم إنشاء إصدار جديد لكل تحديث أو تطوير.\*\*</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- توثيق جميع التعديلات.</span>

<span dir="ltr"></span><span dir="rtl">- تسهيل التعاون بين فرق
العمل.</span>

<span dir="ltr"></span><span dir="rtl">- منع فقدان البيانات.</span>

<span dir="ltr"></span><span dir="rtl">- إمكانية الرجوع لأي إصدار
سابق.</span>

<span dir="ltr"></span><span dir="rtl">- تتبع تطور المشروع خطوة
بخطوة.</span>

<span dir="ltr">\# Version Types</span> <span dir="rtl">(أنواع
الإصدارات)</span>

<span dir="ltr"></span>

<span dir="ltr">\## 1. Major Version</span>

<span dir="ltr"></span>

<span dir="rtl">مثال:</span>

<span dir="ltr"></span>

<span dir="ltr">\`\`\`text</span>

<span dir="ltr">v1.0 → v2.0</span>

<span dir="ltr"></span>

<span dir="rtl">يستخدم عند:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- إضافة أنظمة جديدة.</span>

<span dir="ltr"></span>

<span dir="ltr">\## 2. Minor Version</span>

<span dir="ltr"></span>
