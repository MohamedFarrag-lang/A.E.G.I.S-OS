<span dir="ltr">\# 1. Design Tokens</span>

<span dir="ltr"></span><span dir="rtl">(متغيرات التصميم)</span>

<span dir="ltr"></span>

<span dir="ltr">Design Tokens</span> <span dir="rtl">هي القيم الأساسية
التي يستخدمها النظام بالكامل.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Colors</span>

<span dir="ltr">• Typography</span>

<span dir="ltr">• Spacing</span>

<span dir="ltr">• Radius</span>

<span dir="ltr">• Shadows</span>

<span dir="ltr">• Borders</span>

<span dir="ltr">• Icons</span>

<span dir="ltr">• Motion</span>

<span dir="ltr">• Opacity</span>

<span dir="ltr">• Z-Index</span>

<span dir="ltr"></span>

<span dir="rtl">جميع المكونات تعتمد على</span> <span dir="ltr">Design
Tokens</span> <span dir="rtl">ولا تستخدم قيماً عشوائية.</span>
