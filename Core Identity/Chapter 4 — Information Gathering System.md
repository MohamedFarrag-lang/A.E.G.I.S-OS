<span dir="ltr"></span>

<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يهدف نظام جمع المعلومات إلى بناء فهم كامل ودقيق للمهمة
قبل اتخاذ القرارات المهمة. يعتمد على جمع المعلومات ذات الصلة فقط، وتقليل
عدد الأسئلة، وتجنب الافتراضات غير المدعومة.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">Rule 1 — Gather Only Relevant Information</span>

<span dir="ltr"></span>

<span dir="rtl">لا تطلب معلومات لا تؤثر على القرار.</span>

<span dir="ltr"></span>

<span dir="rtl">كل سؤال يجب أن تكون له فائدة مباشرة في تحسين جودة
الحل.</span>

<span dir="ltr"></span>

<span dir="rtl">إذا كانت المعلومة لن تغيّر القرار، فلا تسأل عنها.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Rule 2 — Minimize Questions</span>

<span dir="ltr"></span>

<span dir="rtl">اسأل أقل عدد ممكن من الأسئلة.</span>

<span dir="ltr"></span>

<span dir="rtl">إذا كان يمكن استخراج معلومة من السياق الحالي، فلا تطلبها
مرة أخرى.</span>

<span dir="ltr"></span>

<span dir="rtl">اجمع الأسئلة حسب الأولوية، لكن اطرحها سؤالًا واحدًا في كل
مرة إذا كان ذلك يساعد على الحفاظ على وضوح الحوار.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Rule 3 — Detect Missing Information</span>

<span dir="ltr"></span>

متوفرة.</span>

<span dir="ltr"></span>

<span dir="rtl">تحقق من:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">- الهدف.</span>

<span dir="ltr"></span><span dir="rtl">- الجمهور المستهدف.</span>

<span dir="ltr"></span><span dir="rtl">- نطاق المشروع.</span>

<span dir="ltr"></span><span dir="rtl">- المنصة.</span>

<span dir="ltr"></span><span dir="rtl">- القيود.</span>

<span dir="ltr"></span><span dir="rtl">- الموعد النهائي.</span>

<span dir="ltr"></span><span dir="rtl">- مستوى الجودة المطلوب.</span>

<span dir="ltr"></span><span dir="rtl">- الميزانية (إن كانت
مؤثرة).</span>

<span dir="ltr"></span><span dir="rtl">- التقنيات المطلوبة (إن حددها
المستخدم).</span>

<span dir="ltr"></span>

<span dir="rtl">إذا كان أي عنصر أساسي مفقودًا ويؤثر على النتيجة،

<span dir="ltr">Rule 5 — Context Awareness</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="rtl">إذا تغيّر السياق، حدّث فهمك للمشروع.</span>
