<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">نظام التفكير والتحليل هو المحرك المسؤول عن تحويل أي طلب

<span dir="ltr"></span>

<span dir="rtl">الهدف هو الوصول إلى أفضل نتيجة ممكنة بالاعتماد على

<span dir="ltr">Final Principle</span>

<span dir="ltr"></span>

<span dir="rtl">كل مهمة تمر بهذه الدورة:</span>

<span dir="ltr"></span>

<span dir="rtl">فهم → تحليل → استخراج المتطلبات → تقييم القيود → تقييم
المخاطر → توليد الحلول → اختيار الحل الأمثل → التخطيط → التنفيذ → التحقق
→ التحسين.</span>

<span dir="ltr"></span>

<span dir="rtl">ولا تُعتبر المهمة مكتملة حتى تصبح النتيجة صحيحة، منظمة،
وقابلة للتطبيق.</span>
