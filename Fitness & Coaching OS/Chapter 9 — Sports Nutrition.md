<span dir="ltr">Chapter 9 — Sports Nutrition</span>

<span dir="ltr"></span>

<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل \*\*</span><span dir="ltr">Sports
Nutrition</span><span dir="rtl">\*\* المرجع المتخصص للتغذية الرياضية
داخل \*\*</span><span dir="ltr">Fitness & Coaching
OS</span><span dir="rtl">\*\*، حيث يركز على كيفية توظيف التغذية لدعم
الأداء الرياضي، وزيادة القوة، وبناء العضلات، وتسريع الاستشفاء، وتحسين
القدرة على التدريب والمنافسة.</span>

<span dir="ltr"></span>

<span dir="rtl">تعتمد التغذية الرياضية الاحترافية على توقيت العناصر
الغذائية، وإدارة الطاقة، وتعويض السوائل والإلكتروليتات، واستخدام
المكملات الغذائية المبنية على الأدلة العلمية، بما يتناسب مع نوع الرياضة،
وشدة التدريب، وأهداف الرياضي.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">\*\*"Train hard, recover smarter, and fuel with
purpose."\*\*</span>

<span dir="ltr"></span>

<span dir="rtl">التغذية الرياضية ليست مجرد تناول الطعام، بل هي
استراتيجية علمية تهدف إلى تحقيق أقصى أداء ممكن داخل وخارج
التمرين.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 1. Pre Workout Nutrition</span>

<span dir="ltr"></span>

<span dir="rtl">تهدف التغذية قبل التمرين إلى توفير الطاقة وتحسين
الأداء.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Complex Carbohydrates</span>

<span dir="ltr">• Lean Protein</span>

<span dir="ltr">• Adequate Hydration</span>

<span dir="ltr">• Electrolytes</span> <span dir="rtl">(عند
الحاجة)</span>

<span dir="ltr"></span>

<span dir="rtl">أفضل الممارسات:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• تناول وجبة متوازنة قبل التمرين
بوقت مناسب.</span>

<span dir="ltr"></span><span dir="rtl">• تجنب الأطعمة الثقيلة أو صعبة
الهضم.</span>

<span dir="ltr"></span><span dir="rtl">• الحفاظ على الترطيب.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 2. Post Workout Nutrition</span>

<span dir="ltr"></span>

<span dir="rtl">تهدف التغذية بعد التمرين إلى دعم الاستشفاء وإعادة بناء
العضلات.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• High-Quality Protein</span>

<span dir="ltr">• Carbohydrates</span>

<span dir="ltr">• Fluids</span>

<span dir="ltr">• Electrolytes</span>

<span dir="ltr"></span>

<span dir="rtl">أهدافها:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• تحفيز تخليق البروتين
العضلي.</span>

<span dir="ltr"></span><span dir="rtl">• تعويض الجليكوجين.</span>

<span dir="ltr"></span><span dir="rtl">• تقليل الإجهاد.</span>

<span dir="ltr"></span><span dir="rtl">• تسريع الاستشفاء.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 3. Carb Cycling</span>

<span dir="ltr"></span>

<span dir="ltr">Carb Cycling</span> <span dir="rtl">هو تنظيم كمية
الكربوهيدرات حسب شدة التدريب والأهداف.</span>

<span dir="ltr"></span>

<span dir="rtl">قد يشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• High Carb Days</span>

<span dir="ltr">• Moderate Carb Days</span>

<span dir="ltr">• Low Carb Days</span>

<span dir="ltr"></span>

<span dir="rtl">الاستخدامات:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• دعم الأداء.</span>

<span dir="ltr"></span><span dir="rtl">• إدارة الطاقة.</span>

<span dir="ltr"></span><span dir="rtl">• التحكم في الوزن.</span>

<span dir="ltr"></span><span dir="rtl">• بعض خطط المنافسات.</span>

<span dir="ltr"></span>

<span dir="rtl">يجب أن يُطبق وفق احتياجات الفرد وليس كنظام ثابت
للجميع.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 4. Peak Performance Nutrition</span>

<span dir="ltr"></span>

<span dir="rtl">تهدف تغذية الأداء إلى تحقيق أعلى مستوى من الكفاءة
البدنية والذهنية.</span>

<span dir="ltr"></span>

<span dir="rtl">تركز على:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• الطاقة الكافية.</span>

<span dir="ltr"></span><span dir="rtl">• جودة المغذيات.</span>

<span dir="ltr"></span><span dir="rtl">• توقيت الوجبات.</span>

<span dir="ltr"></span><span dir="rtl">• الترطيب.</span>

<span dir="ltr"></span><span dir="rtl">• الاستشفاء.</span>

<span dir="ltr"></span>

<span dir="rtl">الأداء المتميز يبدأ بخطة غذائية متوازنة ومستدامة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 5. Recovery Nutrition</span>

<span dir="ltr"></span>

<span dir="ltr">Recovery Nutrition</span> <span dir="rtl">هي التغذية
التي تدعم استعادة الجسم بعد التدريب.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Protein</span>

<span dir="ltr">• Carbohydrates</span>

<span dir="ltr">• Fluids</span>

<span dir="ltr">• Electrolytes</span>

<span dir="ltr">• Antioxidant-Rich Foods</span>

<span dir="ltr"></span>

<span dir="rtl">تساعد على:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• إصلاح الأنسجة.</span>

<span dir="ltr"></span><span dir="rtl">• تقليل التعب.</span>

<span dir="ltr"></span><span dir="rtl">• تحسين الجاهزية للحصة
التالية.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 6. Contest Nutrition</span>

<span dir="ltr"></span>

<span dir="rtl">تغذية المنافسات تختلف حسب نوع الرياضة والمرحلة.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• إدارة السعرات.</span>

<span dir="ltr"></span><span dir="rtl">• التحكم بالكربوهيدرات.</span>

<span dir="ltr"></span><span dir="rtl">• متابعة الترطيب.</span>

<span dir="ltr"></span><span dir="rtl">• الحفاظ على الكتلة
العضلية.</span>

<span dir="ltr"></span><span dir="rtl">• تنظيم الوجبات قبل وأثناء
المنافسة.</span>

<span dir="ltr"></span>

<span dir="rtl">أي تعديلات متقدمة يجب أن تتم بطريقة علمية وتحت إشراف
مختص عند الحاجة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 7. Supplements</span>

<span dir="ltr"></span>

<span dir="rtl">المكملات الغذائية تستخدم لدعم النظام الغذائي وليس
لاستبداله.</span>

<span dir="ltr"></span>

<span dir="rtl">من أكثر المكملات المدعومة بالأدلة العلمية:</span>

<span dir="ltr"></span>

<span dir="ltr">• Whey Protein</span>

<span dir="ltr">• Creatine Monohydrate</span>

<span dir="ltr">• Caffeine</span>

<span dir="ltr">• Omega-3</span>

<span dir="ltr">• Vitamin D</span> <span dir="rtl">(عند الحاجة)</span>

<span dir="ltr">• Electrolytes</span>

<span dir="ltr"></span>

<span dir="rtl">المبدأ الأساسي:</span>

<span dir="ltr"></span>

<span dir="rtl">المكملات تدعم النتائج، لكنها لا تعوض التدريب أو التغذية
غير الجيدة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 8. Electrolytes</span>

<span dir="ltr"></span>

<span dir="rtl">الإلكتروليتات هي معادن تساعد على تنظيم وظائف
الجسم.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Sodium</span>

<span dir="ltr">• Potassium</span>

<span dir="ltr">• Magnesium</span>

<span dir="ltr">• Calcium</span>

<span dir="ltr"></span>

<span dir="rtl">وظائفها:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• توازن السوائل.</span>

<span dir="ltr"></span><span dir="rtl">• انقباض العضلات.</span>

<span dir="ltr"></span><span dir="rtl">• نقل الإشارات العصبية.</span>

<span dir="ltr"></span><span dir="rtl">• دعم الأداء أثناء التدريب الطويل
أو في الأجواء الحارة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 9. Glycogen</span>

<span dir="ltr"></span>

<span dir="ltr">Glycogen</span> <span dir="rtl">هو الشكل المخزن
للكربوهيدرات داخل العضلات والكبد.</span>

<span dir="ltr"></span>

<span dir="rtl">أهميته:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• المصدر الأساسي للطاقة أثناء
التدريب المكثف.</span>

<span dir="ltr"></span><span dir="rtl">• دعم القوة والتحمل.</span>

<span dir="ltr"></span><span dir="rtl">• تحسين الأداء الرياضي.</span>

<span dir="ltr"></span>

<span dir="rtl">يتم تعويضه من خلال تناول الكربوهيدرات المناسبة بعد
استهلاكه.</span>
