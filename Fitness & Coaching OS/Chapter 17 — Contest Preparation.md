<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل \*\*</span><span dir="ltr">Contest
Preparation</span><span dir="rtl">\*\* المرجع الاحترافي للتحضير

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">\## 2. Cutting Phase</span>

<span dir="ltr"></span>

<span dir="rtl">تهدف مرحلة</span> <span dir="ltr">Cutting</span>
<span dir="rtl">إلى تقليل نسبة الدهون مع الحفاظ على الكتلة
العضلية.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Calorie Management</span>

<span dir="ltr">• Resistance Training</span>

<span dir="ltr">• Cardio Programming</span>

<span dir="ltr">• Recovery Monitoring</span>

<span dir="ltr">• Weekly Adjustments</span>

<span dir="ltr"></span>

<span dir="rtl">الهدف هو الوصول إلى أفضل حالة بدنية بطريقة تدريجية
ومستدامة.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 3. Water Manipulation</span>

<span dir="ltr"></span>

<span dir="rtl">قد يقوم بعض الرياضيين بإجراء تعديلات على تناول السوائل
في المراحل الأخيرة من التحضير.</span>

<span dir="ltr"></span>

<span dir="rtl">هذه الممارسات تختلف باختلاف الرياضي والفئة، ويجب أن تتم
فقط تحت إشراف مختص مؤهل، لأن الإدارة غير الصحيحة للسوائل قد تشكل مخاطر
صحية.</span>

<span dir="ltr"></span>

<span dir="rtl">الأولوية دائمًا هي سلامة الرياضي.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 4. Sodium Strategy</span>

<span dir="ltr"></span>

<span dir="rtl">يؤثر الصوديوم في توازن السوائل ووظائف الجسم
الطبيعية.</span>

<span dir="ltr"></span>

<span dir="rtl">أي تعديلات على تناول الصوديوم قبل المنافسة يجب أن تكون
مبنية على خبرة علمية وتحت إشراف مختص، مع تجنب التغييرات العشوائية أو
المبالغ فيها.</span>

<span dir="ltr"></span>

<span dir="ltr">---</span>

<span dir="ltr"></span>

<span dir="ltr">\## 5. Carb Loading</span>

<span dir="ltr"></span>

<span dir="rtl">قد يستخدم بعض الرياضيين استراتيجيات لتنظيم تناول
الكربوهيدرات قبل المنافسة بهدف تحسين امتلاء العضلات.</span>

<span dir="ltr"></span>

<span dir="rtl">يجب أن تكون هذه الاستراتيجيات:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• فردية.</span>

<span dir="ltr"></span><span dir="rtl">• مجربة مسبقًا.</span>

<span dir="ltr"></span><span dir="rtl">• مبنية على استجابة
الرياضي.</span>

<span dir="ltr"></span><span dir="rtl">• تحت إشراف مدرب أو مختص
مؤهل.</span>

<span dir="ltr"></span>
