<span dir="ltr"></span>

<span dir="ltr">\## 1. Motivation</span>

<span dir="ltr"></span>

<span dir="rtl">التحفيز هو الشرارة التي تبدأ رحلة التغيير.</span>

<span dir="ltr"></span>

<span dir="rtl">ينقسم إلى:</span>

<span dir="ltr"></span>

<span dir="ltr">• Intrinsic Motivation</span>

<span dir="ltr">• Extrinsic Motivation</span>

<span dir="ltr"></span>

<span dir="rtl">يساعد التحفيز على:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• تجاوز الصعوبات.</span>

<span dir="ltr"></span>

<span dir="rtl">لكن التحفيز وحده لا يكفي لتحقيق النجاح طويل

<span dir="ltr"></span>

<span dir="ltr">• Nutrition Adherence</span>

<span dir="ltr">• Sleep Routine</span>

<span dir="ltr">• Time Management</span>

<span dir="ltr"></span>

<span dir="rtl">الانضباط هو ما يحول الأهداف إلى نتائج.</span>

<span dir="ltr"></span>

<span dir="ltr">• Hydration</span>

<span dir="ltr">• Sleep Habits</span>
