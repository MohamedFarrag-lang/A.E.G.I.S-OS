<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل
\*\*</span><span dir="ltr">Bodybuilding</span><span dir="rtl">\*\*
المرجع الاحترافي لرياضة كمال الأجسام داخل
\*\*</span><span dir="ltr">Fitness & Coaching
OS</span><span dir="rtl">\*\*، حيث يركز على بناء الجسم من منظور تنافسي

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">Core Philosophy</span>

<span dir="ltr"></span>

<span dir="ltr">\*\*"Bodybuilding is the science of building muscle,
refining symmetry, and presenting the human physique at its highest
level."\*\*</span>

<span dir="ltr"></span>

<span dir="rtl">النجاح في كمال الأجسام يعتمد على التخطيط طويل المدى،
والانضباط، والتطور المستمر، وليس على الحلول السريعة.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">\## 1. Bodybuilding Principles</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد كمال الأجسام الاحترافي على مجموعة من المبادئ
الأساسية.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Muscle Hypertrophy</span>

<span dir="ltr">• Recovery</span>

<span dir="ltr">• Nutrition</span>

<span dir="ltr">• Consistency</span>

<span dir="ltr">• Individualization</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">\## 2. Training Split</span>

<span dir="ltr"></span>

<span dir="rtl">تقسيم التدريب يساعد على إدارة حجم التدريب
والاستشفاء.</span>

<span dir="ltr"></span>

<span dir="rtl">أشهر الأنظمة:</span>

<span dir="ltr"></span>

<span dir="ltr">• Full Body</span>

<span dir="ltr">• Upper / Lower</span>

<span dir="ltr">• Push / Pull / Legs</span>

<span dir="ltr">• Bro Split</span>

<span dir="ltr">• Arnold Split</span>

<span dir="ltr"></span>

<span dir="rtl">يتم اختيار النظام وفق:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• مستوى الرياضي.</span>

<span dir="ltr"></span><span dir="rtl">• الهدف.</span>

<span dir="ltr"></span><span dir="rtl">• القدرة على الاستشفاء.</span>

<span dir="ltr"></span><span dir="rtl">• عدد أيام التدريب.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">• Resistance Training</span>

<span dir="ltr">• Weekly Adjustments</span>

<span dir="ltr">• Progress Monitoring</span>

<span dir="ltr">• Posing Practice</span>

<span dir="ltr"></span>
