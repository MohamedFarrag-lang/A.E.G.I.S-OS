<span dir="ltr">Mission</span>

<span dir="ltr"></span>

<span dir="rtl">يشكل \*\*</span><span dir="ltr">Exercise
Science</span><span dir="rtl">\*\* الأساس العلمي الذي يفسر كيف يستجيب
جسم الإنسان للتدريب، ولماذا تنجح بعض البرامج التدريبية بينما تفشل أخرى.
فمن خلال فهم فسيولوجيا التمرين، وأنظمة الطاقة، وآليات انقباض العضلات،
وإدارة الحمل التدريبي، يصبح المدرب قادرًا على تصميم برامج تحقق أفضل
النتائج بأمان وكفاءة.</span>

<span dir="ltr"></span>

<span dir="rtl">لا يعتمد التدريب الاحترافي على الخبرة فقط، بل على تطبيق
المبادئ العلمية التي تفسر كيفية إنتاج القوة، وبناء العضلات، وتحسين
الأداء، وزيادة التحمل، وتقليل الإجهاد والإصابات.</span>

<span dir="ltr">\## 1. Exercise Physiology</span>

<span dir="ltr"></span>

<span dir="ltr">Exercise Physiology</span> <span dir="rtl">هو العلم الذي
يدرس كيفية استجابة أجهزة الجسم للنشاط البدني.</span>

<span dir="ltr"></span>

<span dir="rtl">يشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Muscular System</span>

<span dir="ltr">• Cardiovascular System</span>

<span dir="ltr">• Respiratory System</span>

<span dir="ltr">• Nervous System</span>

<span dir="ltr">• Endocrine System</span>

<span dir="ltr"></span>

<span dir="rtl">أهدافه:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• تحسين الأداء.</span>

<span dir="ltr"></span><span dir="rtl">• زيادة الكفاءة البدنية.</span>

<span dir="ltr"></span><span dir="rtl">• فهم التكيفات
الفسيولوجية.</span>

<span dir="ltr"></span><span dir="rtl">• تصميم برامج تدريبية
فعالة.</span>

<span dir="ltr">\## 2. Energy Systems</span>

<span dir="ltr"></span>

<span dir="rtl">يعتمد الجسم على ثلاثة أنظمة رئيسية لإنتاج الطاقة أثناء
التدريب:</span>

<span dir="ltr"></span>

<span dir="ltr">\### ATP-PC System</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• أعلى قوة.</span>

<span dir="ltr"></span><span dir="rtl">• أقصر مدة.</span>

<span dir="ltr"></span><span dir="rtl">• يستخدم في رفع الأوزان الثقيلة
والسرعات القصيرة.</span>

<span dir="ltr"></span>

<span dir="ltr">\### Anaerobic Glycolysis</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• مجهود متوسط إلى مرتفع.</span>

<span dir="ltr"></span><span dir="rtl">• إنتاج سريع للطاقة.</span>

<span dir="ltr"></span><span dir="rtl">• يعتمد على الجلوكوز.</span>

<span dir="ltr"></span>

<span dir="ltr">\### Aerobic System</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• إنتاج طاقة مستمر.</span>

<span dir="ltr"></span><span dir="rtl">• يعتمد على الأكسجين.</span>

<span dir="ltr"></span><span dir="rtl">• مناسب للأنشطة طويلة
المدة.</span>

<span dir="ltr"></span>

<span dir="rtl">كل نوع من التدريب يعتمد بدرجات مختلفة على هذه
الأنظمة.</span>
