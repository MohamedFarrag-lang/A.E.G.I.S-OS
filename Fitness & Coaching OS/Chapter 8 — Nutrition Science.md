<span dir="ltr">\## 1. Nutrition Fundamentals</span>

<span dir="ltr"></span>

<span dir="rtl">التغذية هي العلم الذي يدرس كيفية حصول الجسم على العناصر
الغذائية واستخدامها لدعم جميع وظائفه الحيوية.</span>

<span dir="ltr"></span>

<span dir="rtl">أهداف التغذية:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• توفير الطاقة.</span>

<span dir="ltr"></span><span dir="rtl">• بناء وإصلاح الأنسجة.</span>

<span dir="ltr"></span><span dir="rtl">• دعم الجهاز المناعي.</span>

<span dir="ltr"></span><span dir="rtl">• تحسين الأداء الرياضي.</span>

<span dir="ltr"></span><span dir="rtl">• الحفاظ على الصحة.</span>

<span dir="ltr"></span><span dir="rtl">• الوقاية من الأمراض.</span>

<span dir="ltr"></span>

<span dir="rtl">تعتمد التغذية الناجحة على التوازن والاستمرارية وليس
الحرمان.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">\## 2. Macronutrients</span>

<span dir="ltr"></span>

<span dir="ltr">Macronutrients</span> <span dir="rtl">هي العناصر
الغذائية التي يحتاجها الجسم بكميات كبيرة.</span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">\### Protein</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• بناء العضلات.</span>

<span dir="ltr"></span><span dir="rtl">• إصلاح الأنسجة.</span>

<span dir="ltr"></span><span dir="rtl">• دعم الاستشفاء.</span>

<span dir="ltr"></span>

<span dir="ltr">\### Carbohydrates</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• المصدر الرئيسي للطاقة.</span>

<span dir="ltr"></span><span dir="rtl">• دعم الأداء الرياضي.</span>

<span dir="ltr"></span><span dir="rtl">• إعادة تعبئة الجليكوجين.</span>

<span dir="ltr"></span>

<span dir="ltr">\### Fats</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• إنتاج الهرمونات.</span>

<span dir="ltr"></span><span dir="rtl">• امتصاص الفيتامينات.</span>

<span dir="ltr"></span><span dir="rtl">• دعم وظائف الدماغ.</span>

<span dir="ltr"></span><span dir="rtl">• توفير الطاقة طويلة
المدى.</span>

<span dir="ltr"></span>

<span dir="rtl">التوازن بين هذه العناصر يحدد جودة النظام الغذائي.</span>

<span dir="ltr"></span>

<span dir="ltr">\## 4. Calories</span>

<span dir="ltr"></span>

<span dir="rtl">السعرات الحرارية هي وحدة قياس الطاقة التي يحصل عليها
الجسم من الطعام.</span>

<span dir="ltr"></span>

<span dir="rtl">الحالات الأساسية:</span>

<span dir="ltr"></span>

<span dir="ltr">• Calorie Deficit</span>

<span dir="ltr">• Maintenance Calories</span>

<span dir="ltr">• Calorie Surplus</span>

<span dir="ltr"></span>

<span dir="rtl">إدارة السعرات تعتمد على:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• الهدف.</span>

<span dir="ltr"></span><span dir="rtl">• النشاط البدني.</span>

<span dir="ltr"></span><span dir="rtl">• تكوين الجسم.</span>

<span dir="ltr"></span><span dir="rtl">• معدل الأيض.</span>

<span dir="ltr"></span>

<span dir="rtl">السعرات مهمة، لكن جودة الطعام لا تقل أهمية عنها.</span>

<span dir="ltr">\## 5. Meal Timing</span>

<span dir="ltr"></span>

<span dir="ltr">Meal Timing</span> <span dir="rtl">هو تنظيم مواعيد تناول
الطعام بما يتناسب مع النشاط اليومي والتدريب.</span>

<span dir="ltr"></span>

<span dir="rtl">أفضل الممارسات:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• توزيع البروتين على مدار
اليوم.</span>

<span dir="ltr"></span><span dir="rtl">• تناول وجبة قبل التدريب عند
الحاجة.</span>

<span dir="ltr"></span><span dir="rtl">• الاهتمام بالتغذية بعد
التدريب.</span>

<span dir="ltr"></span><span dir="rtl">• الحفاظ على مواعيد
منتظمة.</span>

<span dir="ltr"></span><span dir="rtl">• تجنب الإفراط في الأكل ليلًا إذا
كان يؤثر على الالتزام.</span>

<span dir="ltr"></span>

<span dir="rtl">الالتزام بإجمالي الاحتياجات اليومية أهم من توقيت وجبة
واحدة.</span>
