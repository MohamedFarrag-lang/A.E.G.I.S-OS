<span dir="ltr">\## 1. Biomechanics</span>

<span dir="ltr"></span>

<span dir="ltr">Biomechanics</span> <span dir="rtl">هو علم دراسة حركة
جسم الإنسان باستخدام مبادئ الفيزياء والميكانيكا.</span>

<span dir="ltr"></span>

<span dir="rtl">يركز على:</span>

<span dir="ltr"></span>

<span dir="ltr">• Force</span>

<span dir="ltr">• Torque</span>

<span dir="ltr">• Levers</span>

<span dir="ltr">• Center of Gravity</span>

<span dir="ltr">• Balance</span>

<span dir="ltr">• Stability</span>

<span dir="ltr">• Momentum</span>

<span dir="ltr">• Mechanical Efficiency</span>

<span dir="ltr"></span>

<span dir="rtl">يساعد علم الميكانيكا الحيوية على:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• تحسين الأداء.</span>

<span dir="ltr"></span><span dir="rtl">• زيادة كفاءة الحركة.</span>

<span dir="ltr"></span><span dir="rtl">• تقليل الإجهاد.</span>

<span dir="ltr"></span><span dir="rtl">• الوقاية من الإصابات.</span>

<span dir="ltr">\## 2. Range of Motion</span>

<span dir="ltr"></span>

<span dir="ltr">Range of Motion (ROM</span><span dir="rtl">) هو مقدار
الحركة التي يستطيع المفصل تنفيذها.</span>

<span dir="ltr"></span>

<span dir="rtl">أنواعه:</span>

<span dir="ltr"></span>

<span dir="ltr">• Active ROM</span>

<span dir="ltr">• Passive ROM</span>

<span dir="ltr"></span>

<span dir="rtl">العوامل المؤثرة:</span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• مرونة العضلات.</span>

<span dir="ltr"></span><span dir="rtl">• شكل المفصل.</span>

<span dir="ltr"></span><span dir="rtl">• الأنسجة الضامة.</span>

<span dir="ltr"></span><span dir="rtl">• العمر.</span>

<span dir="ltr"></span><span dir="rtl">• الإصابات السابقة.</span>

<span dir="ltr"></span>

<span dir="rtl">يجب أن يكون مدى الحركة كافيًا لتحقيق الأداء دون التأثير
على الثبات.</span>

<span dir="ltr">\## 3. Mobility</span>

<span dir="ltr"></span>

<span dir="ltr">Mobility</span> <span dir="rtl">هي قدرة الجسم على الحركة

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Joint Mobility</span>

<span dir="ltr">• Soft Tissue Mobility</span>

<span dir="ltr">• Dynamic Mobility</span>

<span dir="ltr">• Functional Mobility</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr"></span><span dir="rtl">• تحسين الأداء.</span>

<span dir="ltr"></span><span dir="rtl">• تقليل الإصابات.</span>

<span dir="ltr"></span><span dir="rtl">• تحسين التقنية.</span>

<span dir="ltr"></span><span dir="rtl">• زيادة جودة الحركة.</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="ltr">\## 4. Stability</span>

<span dir="ltr"></span>

<span dir="ltr"></span>

<span dir="rtl">تشمل:</span>

<span dir="ltr"></span>

<span dir="ltr">• Core Stability</span>

<span dir="ltr">• Joint Stability</span>

<span dir="ltr">• Dynamic Stability</span>

<span dir="ltr">• Neuromuscular Control</span>

<span dir="rtl">الثبات الجيد يسمح بإنتاج قوة أكبر بكفاءة وأمان.</span>

<span dir="ltr">• Pull</span>

<span dir="ltr">• Rotation</span>

<span dir="ltr"></span><span dir="rtl">• تحديد نقاط الضعف.</span>

<span dir="ltr"></span><span dir="rtl">• تقليل الإصابات.</span>

<span dir="ltr">• Push</span>

<span dir="ltr">• Pull</span>
